# 📝 React To-Do List App

This is a simple To-Do List application built using **React** and **normal CSS**.

Users can:
- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed
